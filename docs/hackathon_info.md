# Hack Nation MIT — 대회 정보

## 개요
- **대회명**: Hack Nation MIT
- **형식**: 24시간 안에 실제 AI 프로덕트를 처음부터 빌드
- **협업**: 글로벌 온·오프라인 허브에서 다국적 팀 협업
- **심사**: World-class 심사단 앞에서 라이브 피치
- **주요 스폰서**: OpenAI, Lovable, Hack-Nation (외)

## 심사 기준 (일반적 AI 해커톤 기준, 공식 확정 시 업데이트)
1. 문제 정의의 명확성 및 임팩트
2. 기술적 난이도와 완성도 (실제 동작하는 데모)
3. AI 기술 활용의 창의성 (모델 선택·프롬프트·파인튜닝)
4. UX / 프로덕트 완성도
5. 피치 전달력 (문제 → 솔루션 → 데모 → 비즈니스)

## 팀 구성 (본인 역할)
- **AI**: harness engineering, CNN·LLM 구조 관리, 실험 트래킹 (WandB + MLflow)
- **ML**: PyTorch·TensorFlow 양쪽 준비
