01 · Data blinding & privacy

Health data is high-sensitivity — de-identification is the first step, not an afterthought.

De-identification — what mcPHASES actually did

Remove direct identifiers

Names, emails, device IDs stripped (HIPAA standard).

Blind the dates

Absolute dates → relative day_in_study; only a weekday / weekend flag kept.

Preserve rawness

No imputation — keeps the data real and reproducible.

Privacy-preserving training

Federated learningData stays on device; only model weights are shared.
Synthetic dataTrain / share on generated data, not real records.
Aggregation / k-anonymityPublish statistics, never individual rows.
Differential privacyAdd noise so individuals can't be re-traced.

02 · Learning structure & methods

Across the literature the shape is consistent: signal → physiology-grounded features → gradient-boosted trees. Deep and statistical models come second.

Feature-based trees — the primary approach

Random forest Empatica · 2019

Skin temp, EDA, IBI, HR → 3-phase classification. 87%, AUROC 0.96.

LightGBM ovulation · 2025

ECG HRV temporal patterns + temperature → ovulation window.

CatBoost estradiol · 2026

Sensor time / morphology features → estradiol level (regression).

XGBoost glucose · 2025

Multimodal wearables → glucose. Trees beat linear models; R² 0.73.

Deep-learning time series

TCN + LSTM multimodal · 2024

Temp, HRV, EDA, sleep as raw sequences → real-time phase prediction. Convolution for short patterns, LSTM for long dependencies.

Statistical time series

ARIMA Frontiers · 2023

Forecasts next-day signals (temp, HR).

Circular statistics 2023

Treats the cycle as circular to analyze phase.

Task framing

Classification

4 phases (menses / late-follicular / ovulation / luteal) or ovulation yes/no — most studies.

Regression

Predict LH / E3G / PdG / estradiol values — possible because mcPHASES has hormone-verified labels.

03 · Analysis & validation

This is what separates a credible result from an inflated one.

Splits & labels

Subject-level split

Same person never in train and test — critical at N=42, or scores inflate.

Hormone-verified labels

mcPHASES uses measured LH/E3G/PdG; most prior work uses noisy self-report.

Regular vs irregular

Reported separately — accuracy drops to 50–80% on irregular cycles.

Metrics & interpretation

Performance

Balanced accuracy, AUROC (up to 0.96), R² (0.73), F1.

Calibration + no-call

Brier + reliability curve; abstain when evidence is weak.

Explainability

SHAP for drivers; feature importance ≠ biological cause; subgroup-equity audit.
